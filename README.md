Как начать писать?
Для начала нужно установить на свой компьютер последнюю версию языка.

в папке header лежит заголовочный файл "Ве_крест_крест.h", который необходимо скачать;
далее нужно создать проект на языке C++, под видом которого долгие годы прятался В††. В папку этого проекта, в которой находятся файлы сборки, переносим файл "Ве_крест.h";
теперь для того, чтобы начать писать, нужно только подключить заголовочный файл при помощи первой и последней для вашего кода ящерской команды

#include "Ве_крест.h"
чтобы использовать новую славянскую структуру данных, её необходимо взять из Сибири, для этого написать строчки кода
